# Risk-Evaluation-of-Fine-Tuned-VLMs
Fine-tuned Qwen2.5-VL-7B and Qwen3-VL-8B with QLoRA on chest X-ray pneumonia classification. Improved F1 by 424% over zero-shot baselines. Engineered 9 custom evaluation metrics revealing that 77% of incorrect predictions were delivered with confident, fluent explanations — exposing a critical safety gap in fine-tuned VLMs. Collaborator: Kaavin
